This repository contains a consolidated Customer Performance Report, designed to analyze how customers, markets, and product segments performed across different regions and divisions. The Excel model includes dashboards, filters, and supporting sheets that make it easy to explore performance from multiple angles.

The workbook includes several dashboards and summary sheets that highlight customer behaviour and business performance.
Key areas covered in the report include:
